EcoSphere is a web platform that helps companies track their ESG (Environmental, Social, and Governance) progress. Instead of using messy spreadsheets, it puts carbon tracking, community activities, and company policies into one clean dashboard. It even uses a "gamification" system with XP and challenges to make sustainability engaging for employees.
It keeps an eye on carbon emissions and monitors whether company departments are hitting their green goals.
It tracks volunteer events, beach cleanups, and team participation.
It stores internal audits, keeps track of company policy compliance, and flags high-risk issues that need fixing.
Employees can join fun sustainability sprints, earn points (XP), and unlock badges for their hard work.
It includes a built-in security setup that emails a 6 digit verification code (OTP) to make sure only real team meambers can create an account.
Pure HTML, CSS, and interactiive JavaScript. No complex frameworks are required.
Node.js to handle data requests and routing.
The database is a simple data.json file. It works like a lightweight, automatic notepad on our computer.
We used nodemailer to connect securely to Gamil to send verification codes.
We have an EcoSphere folder which contains README.md, index.html, data.json, package-lock.json, package.json, server.js file in it.
